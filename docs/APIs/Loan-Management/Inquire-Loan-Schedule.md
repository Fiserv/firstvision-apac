# Inquire Loan Schedule

This API is used to fetch loan schedule details for the current loan data.

## Endpoint

`GET /v1/loans/{accountId}/scheduleDetails`

type: tab
titles: Request, Response, Error
-->

>Should be empty.
>
>***Account Id should be sent as path variable and plan sequence number sent as Query Parameter.***
